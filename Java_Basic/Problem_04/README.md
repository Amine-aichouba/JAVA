## * problem 04 : Arithmetic Operations

Write a Java program to print the results of the following operations.

## * Test Data :
    
    a = -5 + 8 * 6
    
    b = ( 55 + 9 ) % 9
    
    c = 20 + -3*5 / 8
    
    d = 5 + 15 / 3 * 2 - 8 % 3

## * Expected Output :

    a : -5 + 8 * 6 = 43 
    
    b : ( 55 + 9 ) % 9 = 1 
    
    c : 20 + (-3) * 5 / 8 = 19 
    
    d : 5 + 15 / 3 * 2 - 8 % 3 = 13
